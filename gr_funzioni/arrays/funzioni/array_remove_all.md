# array\_remove\_all

## Funzione `array_remove_all`

Restituisce un array con tutti gli elementi del valore passato rimossi.

## Sintassi

* array_remove\_all\(\_array, value_\)

## Argomenti

* _array_ un array
* _value_ i valori da rimuovere

## Esempi

* `array_remove_all(array('a','b','c','b'),'b') â†’ array: 'a','c'`
